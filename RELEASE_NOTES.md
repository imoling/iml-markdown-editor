# iML Markdown Editor v1.6.0 - 极简其表 · 极致内核

本次 v1.6.0 版本是一次里程碑式的更新，我们通过对 AI 交互、图形渲染及系统性能的深度调优，正式确立了 **“极简其表 · 极致内核”** 的产品灵魂。

## ✨ 版本亮点

### 🪄 指令级 AI 助手 (AIPalette)
- **极致净空界面**：移除了冗余的快捷键提示文本，视觉重心完全聚焦于创作本身。
- **实体感交互设计**：为“结合上下文”、“Mermaid”及“SVG”注入了浅色实体背景与动态阴影，交互反馈更具质感。
- **常驻发送确认**：重构了发送按钮逻辑，确保在不同输入状态下映射最清晰的操作意图。

### 🎨 响应式图形引擎
- **自由拖放缩放**：Mermaid 流程图与 SVG 图标现在支持直接在编辑器内 **实时拖拽边缘进行纵向/缩放调整**。
- **等比放大逻辑**：优化了内部组件的同步放大算法，确保图形在尺寸调整时依然保持完美的比例与清晰度。

### ⌨️ 分页式快捷键系统
- **三页式负载均衡**：将快捷键手册划分为“核心文件”、“创作辅助”、“深度表格”三个维度，彻底解决了内容拥挤问题。
- **点状导航**：通过全新的分页指示器提供确定性的查阅指引。

### 📄 专业级导出与全平台支持
- **PDF 渲染修复**：全面解决了 PDF 导出时 SVG 与 Mermaid 图形渲染丢失的问题，确保所见即所得。
- **Windows 版正式加入**：v1.6.0 正式提供 Windows 安装程序（.exe），支持无边框窗口模式与桌面原生交互适配。

## 🛠 技术改进
- 优化了双模式切换时的内存回收机制。
- 刷新了“关于”弹窗的视觉层次与品牌文案。
- 更新致谢标识为 Antigravity AI。
