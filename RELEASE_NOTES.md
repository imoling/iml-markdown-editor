# iML Markdown Editor v1.7.0 - AI 时代的敏捷知识编辑中枢

本次 v1.7.0 版本不仅是一次功能升级，更是一次**产品定位的质变**。我们正式将 iML 从一个“Markdown 编辑器”进化为具备文件深度感知、工作流持久化与智能控制能力的“**敏捷知识管理库 (Knowledge Base)**”。

---

## ✨ 版本亮点：从工具到中枢

### 📁 深度文件管理系统 (Knowledge Sidebar)
- **原生级右键增强**：侧边栏现支持重命名、复制副本、新建子文档及安全推入系统回收站。
- **极客交互快捷键**：支持 `F2` 直接重命名、`Cmd + D` 快速克隆文档逻辑，操作效率提升 300%。
- **星标收藏与近期足迹**：全新 `Recent Files` 逻辑，精准记录您的创作路径；支持 Workspaces 级别的文档星标收藏。

### 🎨 极简智能控制台 (Smart Dashboard)
- **Dashboard 启动页**：当关闭所有标签页时，系统自动唤起具备毛玻璃拟态设计的 Dashboard，并展示基于地理与时间的智能欢迎语。
- **视觉 Badge 体系**：在首页与 About 窗口增加了精致的 v1.7.0 工业级版本标签，彰显品牌正交。
- **Slogan 语义升级**：确立了“**AI 时代的敏捷知识编辑中枢**”这一全新使命。

### 🔄 全局会话持久化 (Session Persistence)
- **零上下文损失**：即使应用冷启动，iML 也会为您自动恢复上次未关闭的所有 Tabs、工作区路径以及精准的滚动条位置（Scroll Position）。
- **无感自动保存**：监听窗口失焦（Blur）与多任务切换事件，实现秒级后台静默存盘，彻底告别“Ctrl+S”焦虑。

### 🛠️ UI/UX 极致调优
- **设置中心重塑**：彻底优化了 `SettingsModal` 在小窗模式下的垂直滚动表现，解决了内容溢出的视觉顽疾。
- **版本通信加固**：重构了 Electron 与渲染进程的版本通信逻辑，解决开发环境下 `process.env` 丢失导致的版本号 fallback 异常。
- **品牌视觉对齐**：统一了 `About` 窗口与 `StartPage` 的视觉语言，复活了经典的“**极简其表 · 极致内核**”八字真言。
