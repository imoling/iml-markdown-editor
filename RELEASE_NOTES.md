# iML Markdown Editor v1.6.0 - 极简其表 · 极致内核

本次 v1.6.0 版本是一次里程碑式的更新，我们通过对 AI 交互、图形渲染及系统性能的深度调优，正式确立了 **“极简其表 · 极致内核”** 的产品灵魂。

## ✨ 版本亮点

### 🪄 指令级 AI 助手 (AIPalette)
- **结合上下文能力**：新增“结合上下文”AI 补充文案能力，支持一键切换是否引入当前文档全量背景。
- **重构发送按钮逻辑**：确保在不同输入状态下映射最清晰的操作意图。
- **模型配置精构**：配置页面 API ENDPOINT 更新为 **BASE URL**，并增加了隐私安全加密提示。

### 🎨 响应式图形引擎
- **AI 生成增强**：实现响应式图形引擎，支持 AI 直接生成可交互的 **Mermaid 图**与 **SVG 图**。
- **自由拖放缩放**：优化内部组件的同步放大算法，确保图形在尺寸调整时依然保持完美的比例与清晰度。
- **PDF 完美导出**：全面解决了导出 PDF 时图形渲染丢失的问题，确保所见即所得。

### 🚀 智能更新系统
- **自动检测机制**：应用启动 3 秒后自动静默检测 GitHub 最新版本。
- **感应式红点提示**：当有新版本时，“帮助”菜单及“检查更新”项将亮起**极致美感的呼吸红点**，无扰提醒。

### 🖥️ Windows 版正式首发
- **突破性更新**：v1.6.0 是 iML Markdown Editor **首次正式支持 Windows 平台**。
- **深度定制安装程序**：全新设计的 **Nordic Clarity** 品牌安装向导，支持 x64 与 arm64 双架构。
- **子窗口交互补全**：修复了“关于”、“快捷键”等子窗口在 Windows 下缺少关闭控制的问题，实现完美的原生闭环。

## 🛠 技术改进
- 优化了双模式切换及实时渲染时的内存回收机制。
- 移除了冗余的快捷键提示文本，让视觉重心完全聚焦于内容。
- 刷新了“关于”弹窗的视觉层次与品牌文案，更新致谢标识为 Antigravity AI。
- **分页式手册**：将快捷键手册划分为“核心文件”、“创作辅助”、“深度表格”三个维度，彻底解决了内容拥挤。
